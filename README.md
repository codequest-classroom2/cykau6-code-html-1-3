### 📖 **Step-by-Step Guide**

#### Step 1: Understanding the Structure
Every HTML page needs a proper foundation. Think of it like building a house:

```
┌─────────────────────┐
│  <!DOCTYPE html>    │  ← The permit
├─────────────────────┤
│  <html>             │  ← The house
│  ┌───────────────┐  │
│  │ <head>        │  │  ← Blueprints
│  │   <title>     │  │  ← House name
│  │ </head>       │  │
│  ├───────────────┤  │
│  │ <body>        │  │  ← Living space
│  │   <h1>        │  │  ← Big sign
│  │   <p>         │  │  ← Furniture
│  │ </body>       │  │
│  └───────────────┘  │
│  </html>            │
└─────────────────────┘
```

#### Step 2: Try This Example
```html
<!DOCTYPE html>
<html>
<head>
    <title>My Awesome Page</title>
</head>
<body>
    <h1>Welcome to My Website! 🌟</h1>
    <p>This is my first paragraph. I'm learning HTML!</p>
    <p>This is another paragraph. Look at me go! 🚀</p>
</body>
</html>
```
